HTML

4 Components

Structure in HTML

Code should be structured to help with Organization

  • HTML elements are either inline or block
  • HTML has a "tree-like structure" - elements can have other elements inside of them
  • Each Element corresponds to a BOX on the webpage
  • Programmers indent these elements to correspond with the box

Using CSS to add STYLE

CSS - Cascading Style Sheets

  • CSS is used to control the style of the corresponding HTML page
  • HTML Elements are given class names, and then style is applied to that class

Avoiding Repetition

  • Using classes to avoid repetition helps to avoid errors
  • Using classes also makes it easier to make changes to style in one place

Intro to "Serious" Programming

What is Programming

  • Programming is the core of computer science
  • A computer is a machine that can execute a program
  • A program is a specific sequence of steps for the computer to follow without any thought
  • Programmers indent these elements to correspond with the box

Programming Language

  • A programming language is designed to make it easy for humans to read and write programs that can be executed by a computer
  • Programming languages have strict grammar rules.
  • PYTHON is an example of a programming language.
  • Some different elements in Python include Expressions, Variable, Strings, and Numbers

Variables and Strings

  • Variables are useful for storing data to be used in expressions
  • Variables can be changed easily without having to change the expressions
  • Strings are a sequence of characters surrounded by quotes
  • Strings can be Indexed so that characters or sub-sequences can be extracted from them

Input-Functions-Output

  • Use Procedures(or Functions) to make repeated tasks easier
  • Procedures map inputs to outputs
  • If a function must have a return statement to provide and output

Decisions and Repetition

  • Comparison of numbers outputs a Boolean Value(ie: True or False), not a number.
  • IF Statements can be used with comparisons to make decisions on what code to run.
  • Loops are a way to do things over and over again.

Debugging Strategies

  • Examine Error Messages
  • Work from a working example
  • Check intermediate results. Add print statements in your code to check status
  • Keep and compare old versions

Structured Data: Lists and For Loops

Lists

  • String is a sequence of characters.
  • List is a sequence of anything.
  • A Nested Lists are lists within a list
  • List can be appended to.

Mutation and Aliasing

  • Mutation is the ability to change the value of the elements of a list
  • Strings are not mutable.
  • Aliasing is having different ways to refer to the same object.

for Loops

  • for Loops provide a simple way to cycle through elements in a list.
